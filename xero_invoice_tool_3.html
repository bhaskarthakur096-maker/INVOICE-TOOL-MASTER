🔧 Xero API setup — one-time
  • Create a Xero app — go to developer.xero.com/app/manage → New app → Web app.
  • Set redirect URI — copy the URI shown when you click "Connect Xero" and add it to your Xero app's Redirect URIs.
  • Paste credentials — enter your Client ID and Client Secret in the Connect Xero dialog.
  • Authorise — click "Authorise with Xero" and complete the login in the popup.
  • Done! — drop multiple invoice PDFs/images into the queue and hit Process All.
